# Aditya Ananth Portfolio

A clean one-page portfolio website for a fresher data analyst.

## Files
- `index.html` — full portfolio page with responsive styling.

## Quick edit checklist
1. Replace the LinkedIn link if your exact URL is different.
2. Add screenshots/GIFs for Power BI dashboards later if available.
3. Add a downloadable resume PDF by placing it in this folder and linking it from the hero section.

## Free hosting options
- GitHub Pages: best because your projects are already on GitHub.
- Netlify: easiest drag-and-drop deployment.
- Vercel: good if you later convert this into a React portfolio.
